# Scholarship Management System

## Project Overview
Scholarship Management System is a web-based application developed to simplify the scholarship application and management process. The system allows students to register, apply for scholarships online, and track their application process. It also provides an admin panel to manage applications efficiently.

## Features

### Student Module
- Student Registration and Login
- Online Scholarship Application Submission
- Form Validation
- View Application Status

### Admin Module
- Secure Admin Login
- View Student Applications
- Approve or Reject Applications
- Manage Scholarship Records

## Technologies Used
- ASP.NET Web Forms
- C#
- HTML
- CSS
- SQL Server


## System Workflow
1. Student registers in the system.
2. Student logs in and fills scholarship application form.
3. Application data is stored in SQL Server database.
4. Admin reviews submitted applications.
5. Admin approves or rejects applications.
6. Student can check application status.

## Database
Database is developed using SQL Server for storing:
- Student Details
- Scholarship Applications
- Login Information
- Application Status

## Future Enhancements
- Email Notifications

- Document Upload

## Installation Steps
1. Clone the repository.
2. Open project in Visual Studio.
3. Configure SQL Server connection string.
4. Run database scripts.
5. Build and run the project.
